===========================================================
spiderpig resource support
===========================================================

- resource/ folder with files containing data for spiderpig
- let irc clients add/edit/delete
- simple api for developers to use resources in new modules

syntax:
- pig: add to trafikaten.csv: oslo s; 32588127
- pig: add to bootycall.txt:
<pig> what to add?
- pig: narrow eyebrows

- pig: remove from bootycall.txt: ass
- pig: edit trafikanten.txt:
<pig> what to edit?
- pig: nostril
<pig> what to change it to:
- pig: nostrils

- pig: create file trafikanten.txt
- pig: show me trafikanten.txt (shows 20 lines)
- pig: show me resource files

api:
import pigFood
pigFood.getRandomLine('bootycall.txt')

pigFood.getCvs('trafikanten.csv')
stationId = pigFood.getValue('oslo s')
